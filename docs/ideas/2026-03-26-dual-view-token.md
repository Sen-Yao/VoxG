# 双视图 Token：Cross-channel + Channel-wise

## 核心想法

模仿 RHO 的双视图学习，在 VoxGFormer 中同时使用 Cross-channel 和 Channel-wise 两种 Token，捕捉不同粒度的传播模式。

## 直觉来源

1. **RHO 论文**：正常节点 homophily 分布很广，需要自适应滤波
2. **通道异质性**：不同特征通道可能有不同的传播行为
3. **信息互补**：Cross-channel 捕捉协同，Channel-wise 捕捉独立

## 方案设计

### Cross-channel View（现有）

```python
T_cross = hop2token(X)  # [N, K+1, D]
```

### Channel-wise View（新增）

```python
# 方案A：每个通道单独传播
for i in range(D):
    T_channel_i = hop2token(X[:, i:i+1])  # [N, K+1, 1]

# 方案B：通道分组
for g in range(G):
    T_group_g = hop2token(X[:, g*size:(g+1)*size])
```

### 融合策略

| 策略 | 复杂度 | 说明 |
|------|--------|------|
| 早期融合 | 低 | 直接拼接 |
| 并行双编码器 | 高 | 两套 Transformer |
| 通道分组 | 中 | 折中方案 |
